## 🚗 交通策略（一人・唔包車・Kakao T + 巴士）

**得！一個人純 Kakao T + 公車可行，重點係「巴士做骨幹、Kakao T 補最後一程」。**

| 場景 | 方法 |
|---|---|
| 機場 ⇄ 西歸浦酒店 | **機場巴士 600 號**（終點站＝KAL！₩5,000・~80分・每15–20分・機場5號閘口） |
| 西歸浦市區 / 中文區 | **Kakao T** 短程，平又快 |
| 西部（涯月/挾才/翰林） | **沿岸巴士 202** 做骨幹（班次密）＋ Kakao T 補尾程 |
| 東部（城山/萬丈窟/月汀里） | **巴士 201 / 101** 沿線 ＋ Kakao T |
| 中部（Osulloc/山君不離/漢拿山） | 最荒：漢拿山登山口 **281 / 240**；其餘景點班次疏，Naver 查時間 |
| Uber | ❌ 濟州市以外幾乎冇車，唔好靠 |

**令一人純 Kakao T 行得通嘅 5 招**
1. **巴士做骨幹**（西岸202、東岸201/101），Kakao T 只補「最後一公里」。
2. **落車即同司機攞 KakaoTalk／電話**，走時直接 call 返佢嚟接（濟州好多司機肯）。
3. 叫車揀 **「대기（等候）」**，請司機喺荒點等你（畀等候費）＝臨時迷你包車，只喺有需要先用。
4. **每個遠程日最多 2–3 個荒點**，就算等車都唔會亂晒。
5. **唔好喺荒點留到黃昏/收場**先走，嗰陣最易冇車。

- App：Kakao T、Naver Map、Papago、CatchTable。[2][6][8]
- 濟州市⇄西歸浦的士 ~₩40,000–50,000；巴士600只要 ₩5,000。
