## 📅 逐日行程

### Day 1 — 6/20（六）抵達 + 西歸浦安頓【輕鬆】
通宵機會攰，第一日唔好排太密。
- 06:30 落機 CJU，攞行李
- 早餐（機場附近）：올래국수 Ollae Guksu 豬肉麵 / 돌담흑돼지 機場店黑豬肉 [3]
- 約 09:00–10:00 搭 **巴士600（5號閘口）→ Seogwipo KAL Hotel**（~80分，₩5,000）
- 到酒店寄行李，食午餐
- 下午（步行 / 短程 Kakao T，近酒店）：正房瀑布 정방폭포 → 獨立岩 Oedolgae 외돌개 [7]
- 15:00 Check-in，回房抖一抖
- 黃昏：天地淵瀑布 천지연폭포（夜間點燈，近酒店）[7]
- 晚餐：西歸浦每日偶來市場 서귀포매일올레시장 + 帶魚 네거리식당 Negeori Sikdang [3][7]

### Day 2 — 6/21（日）西歸浦南岸 + 中文觀光區【步行/短程 Kakao T】
全日都喺南岸，唔使長途，車程短。
- 上午：柱狀節理帶 주상절리 Jusangjeolli Cliff → 中文海水浴場 중문색달해변 [7]
- 午餐（中文區）：색달식당 Saekdal Sikdang 帶魚 / 刀魚湯 [3]
- 下午（任揀）：山茶花之丘 동백포레스트/카멜리아힐 Camellia Hill（~20分車）/ 泰迪熊博物館 / 小人國 [7]
- 黃昏：返獨立岩或中文海邊睇日落
- 晚餐：西歸浦市區（國수바다 Guksubada 麵 / 호정이네 帶魚）[3]

### Day 3 — 6/22（一）東部一日遊【建議包私人的士/包車】
最遠（離西歸浦 50–70 分），返程車最難 call → 包車一日最穩。
- 城山日出峰 성산일출봉（早去人少）[2][3][7]
- 涉地可支 섭지코지 Seopjikoji [7][11]
- 午餐：명진전복 Myeongjin 鮑魚砂鍋飯 [3][12]
- 萬丈窟 만장굴 Manjanggul（UNESCO 熔岩洞，入面涼）[11]
- 月汀里海邊 월정리 Woljeongri — 海景 cafe [7][17]
- （想去 **牛島** 嘅話，東部要用足一整日：城山搭船去牛島租單車+海景cafe，咁就跳過萬丈窟）[3][7][12]

### Day 4 — 6/23（二）中部 — 漢拿山 或 茶+火山口
⚠️ 週二好多市場/食肆休息（東門市場週二休），所以中部行山/景點啱。
- **方案 A（行山）**：漢拿山 한라산（城板岳/觀音寺線，需網上預約、極早出發、時間管制嚴）[2][7][11]
- **方案 B（輕鬆）**：O'Sulloc 茶博物館 오설록（抹茶）→ 山君不離 산굼부리 Sangumburi 火山口 → Eco Land 森林小火車 [7][11]
- 中部景點疏，Kakao T 返程慢 → 同樣建議包車 / 半日tour

### Day 5 — 6/24（三）西部一日遊【建議包車 / 沿岸巴士202】
- 涯月咖啡一條街 애월 + 涯月海岸（海景cafe密度最高）[7][11][12]
- 挾才海水浴場 협재해변 + 翰林公園 한림공원 [7]
- 오드랑 베이커리 Odurang Bakery 蒜蓉爆漿法包（早去，賣完即止）[1][3][12][15]
- 涯月鮭魚本店 三文魚刺身（11:00–21:00，週日休）[20]
- 黃昏：挾才/涯月睇日落

### Day 6 — 6/25（四）濟州市 / 北部 購物日
- 搭 **巴士600** 或 Kakao T 返濟州市
- London Bagel Museum 런던베이글뮤지엄（要排隊，一開門就去）[11][13][16]
- 東門市場 동문재래시장 街頭小食 + 手信（07:00–21:00）[3][7][17]
- 新羅 / 樂天免稅店 + 步行街購物 [7]
- 龍頭岩 용두암 [17]
- 黑豬肉晚餐：만복흑돼지 Manbok（#1）/ 칠돈가 Childonga [3]
- 尾班 **巴士600** 由機場 22:50 開返西歸浦（或 Kakao T ~₩45,000）

### Day 7 — 6/26（五）機動 / 手信 / 休息日
- 西歸浦每日偶來市場 補手信：柑橘朱古力/果醬/果凍、馬格利（柑橘製品要買加工嘅，鮮果帶唔到）[3][16][21]
- 落雨後備：Zarolo Museum 沉浸式光影展 [14]；或重遊最鍾意嘅景點 / 酒店泳池+spa 抖
- 執行李，早瞓
- **⚠️ 今晚預約好聽朝去機場嘅私人接送 / 叫酒店幫手安排的士**（4–5am Kakao T 唔可靠）

### Day 8 — 6/27（六）回程
- ~04:30 離開酒店（私人接送 ~60分 → CJU，約 ₩45,000–60,000）
- ~05:30 到機場，國際線 check-in（HK Express 建議起飛前 90 分到）
- **07:30 UO699 → 香港 09:40**
